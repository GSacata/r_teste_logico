ENUNCIADO
4.	Dado o array [9, 2, 3, 1, 4] encontre todos os números que estão faltando para completar o intervalo 0 a n, onde n é o maior número dentro do array. Adicione os números faltando dentro do array. 


PASSOS
.pegar todos os números existentes no array.
.guardar o maior número em uma variável.
.Usar o maior número como limite do for.
.Ter um array que tenha os números inseridos.



TESTES
[ ] Array proposto.
[ ] Array com outro número maior que o original.
[ ] Array com número repetido (deve não checar números repetidos).
[ ] Array com número negativo.
[ ] Não-números (ex.: letra) no array.
